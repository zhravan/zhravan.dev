# Code Language Badges Demo

This post showcases the **Code Language Badge** plugin - a new feature that displays language identifiers on code blocks!

## Features

**Multiple Styles**: Choose from default, minimal, or pill styles
ðŸ“ **Flexible Positioning**: Position badges on top-left or top-right
**60+ Languages**: Pre-configured support for most popular languages

## Configuration

You can customize the appearance in `config/plugins.yaml`:

```yaml
code-language-badge:
  enabled: true
  position: "top-right"    # or "top-left"
  showIcon: true           # Show emoji icons
  style: "default"         # "default", "minimal", or "pill"
```

Try hovering over the code blocks to see the copy button too!

# Building Modern Web Apps with Next.js

A comprehensive talk about the latest features in Next.js 15 and how to build performant, modern web applications.

## Talk Overview

In this presentation, I covered:

- **App Router Architecture** - Understanding the new file-based routing
- **Server Components** - Benefits and use cases
- **Data Fetching Patterns** - Streaming, suspense, and parallel data loading
- **Performance Optimization** - Core Web Vitals and best practices

## Key Takeaways

### 1. Server Components Are Powerful

```typescript
// Server Component - runs on the server
export default async function BlogPost({ params }) {
  const post = await getPost(params.slug);
  return <Article post={post} />;
}
```

Server Components allow you to:
- Fetch data directly without APIs
- Keep sensitive code on the server
- Reduce client bundle size

### 2. Streaming for Better UX

Use `<Suspense>` to stream content as it's ready:

```typescript
<Suspense fallback={<Skeleton />}>
  <SlowComponent />
</Suspense>
```

### 3. Metadata API

The new Metadata API makes SEO easy:

```typescript
export const metadata = {
  title: 'My Page',
  description: 'Page description',
  openGraph: { ... }
};
```
